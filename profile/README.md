# deltabeam

### Exploration of DEX on ERC-20 Chains — Moonbeam GLMR Parachain

> 8 repositories focused on DEX infrastructure on the Moonbeam (GLMR) parachain, including StellaSwap core, Uniswap V2, DeFi analytics, and Substrate development tools.

---

## What This Is

deltamoonbeam collects the source code for building decentralized exchanges on Moonbeam — the Ethereum-compatible parachain on Polkadot. Moonbeam provides full EVM compatibility with the added interoperability of the Polkadot relay chain, making it ideal for cross-chain DEX deployment.
